RPLearn

Laravel 12 MySQL Tailwind CSS Alpine.js Vite 7 Blade

RPLearn is a web-based Learning Management System (LMS) platform designed specifically for school environments. This platform provides features such as content likes & saves, a terminology dictionary (dictionary), and FAQs / Q&A to support a more interactive and structured learning experience. Built with the TALL stack (Tailwind, Alpine.js, Laravel 12, Blade) and supported by Vite 7 as a modern build tool for fast asset compilation. In this team project, I contributed as a Fullstack Developer with a focus on the admin panel managing all system data, content, and configurations.
