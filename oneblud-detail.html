OneBLUD

Laravel 12 PHP 8.3 MySQL 8.0 Tailwind CSS Alpine.js Vite WhatsApp Gateway Realtime Notification WhatsApp API Axios Fonnte API DomPDF Maatwebsite Excel

OneBLUD is an integrated service booking web application designed for vocational schools (SMK) as well as public institutions. This platform allows users to book various services ranging from futsal fields, cafeterias, vehicle workshops, to AC servicing in a single integrated system. Built on top of Laravel 12 with the TALL stack (Tailwind, Alpine.js, Laravel, Blade), this project includes an authentication system, WhatsApp notifications (Fonnte), PDF & Excel report exports, and a Docker-based development environment (Laravel Sail). In this team project, I contributed as a Fullstack Developer with a focus on the futsal booking system, from database design, backend logic, to a responsive frontend display.
